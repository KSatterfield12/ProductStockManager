Notes: The Product class encapsulates product details. 
The InventoryManager class contains all methods for managing products and persistence. 
The Program class contains the main loop and delegates menu choices to InventoryManager.

-- Variables Plan --
InventoryManager.products: List<Product>

Temporary inputs: string name, decimal price, int stockQuantity, int change

Menu control: string choice, bool exit


-- Methods Plan --
AddProduct: Input name, price, stockQuantity. Validates duplicates, creates product, saves to file.

UpdateStock: Input product name, stock change. Finds product, adjusts stock, prevents negative, saves to file.

ViewProducts: Displays list of products, allows sorting by name, price, or stock.

RemoveProduct: Input product name. Finds product, removes from list, saves to file.

SearchProduct: Input product name. Finds product, displays details.

SaveProducts: Serializes product list to JSON file.

LoadProducts: Deserializes product list from JSON file at startup.


-- Control Structures --
Switch-case: menu navigation in Program.Main.

If-else: validation (duplicate product, invalid input, product exists, stock non-negative).

While loop: keeps program running until exit.

Foreach loop: iterates through product list in ViewProducts.

LINQ methods (Any, Find, OrderBy): used for searching, duplicate prevention, and sorting.


-- Loops in Detail --
Main loop: while (!exit) → keeps menu active in Program.

Product listing loop: foreach (var product in products) → displays inventory in ViewProducts.

Sorting loop: LINQ OrderBy used to sort products by name, price, or stock.
